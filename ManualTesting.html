Manual Testing

Steps I follow during Manual Testing:

  1. Analyze Business Requirements Document.Also,understand system requirements and data mapping specifications.
  2. Design and develope test strategy, test plans, test Cases that cover all the requirements mentioned in the documentation
  3. Review and baseline the test cases with Team Lead, Client (as applicable)
  4. Execute the test cases on the TestRail
  5. Report bugs.
  6. Once bugs are fixed, again execute the failing test cases to verify they pass.
