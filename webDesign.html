Web Design

Steps I follow while Design Web Application:

  1. Source an idea, doing market research then discuss my research with Team lead, Stakeholders and cilent and define functionality.
  2. Sketch web app, plan workflow and prepare wireframe to take validation.
  3. Architect Database using SQL queries with MySQL. Develop Frontend using HTML, CSS and JavaScript then build backend.
  4. Host my web app with domain.
  5. Deploy web app code using continuous integration tools such as Git & Jenkins.
